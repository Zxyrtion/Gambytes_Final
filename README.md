# 🎯 Gambytes – Gambling Recovery & Financial Management System

## 📌 Project Description

Gambytes is a web-based application designed to help individuals struggling with online gambling manage their finances, monitor behavior, and support recovery. The system also allows family members and stakeholders to monitor progress and assist in decision-making.

This project follows the **MVC (Model-View-Controller)** architecture to ensure clean separation of concerns, scalability, and maintainability.

---

## 🎯 Objectives

* Help users track gambling-related expenses
* Provide tools for emotional awareness (mood logs)
* Support financial control through budgeting
* Enable self-exclusion scheduling
* Allow family monitoring and support
* Reduce financial harm caused by gambling

---

## 👥 Target Users

* Gamblers (Primary users)
* Family Members
* Admin Staff
* Case Managers
* Nurses
* Supervisors

---

## ⚙️ Key Features

* 🔐 User Authentication (Login/Register)
* 📅 Scheduling System
* 📄 Contract & Agreement Management
* 💳 Payment & Receipt System
* 🧠 Therapy Progress Monitoring
* 💊 Medication Tracking
* 📈 Evaluation & Discharge System

---

## 🏗️ System Architecture

The system uses **MVC Architecture**:

* **Model** → Handles database operations
* **View** → User interface (UI)
* **Controller** → Handles logic and user requests

---

## 🔄 System Workflow

1. User registers and logs in
2. Books a schedule
3. Reviews and signs contract
4. Makes payment
5. Undergoes therapy phases
6. Progress is monitored
7. Evaluation is conducted
8. User is discharged or extended

## ▶️ How to Run the Project

1. Install XAMPP
2. Place project folder in `htdocs`
3. Start Apache and MySQL
4. Import database (`gambytes.sql`)
5. Open browser:

   ```
   http://localhost/gambytes/public
   ```
